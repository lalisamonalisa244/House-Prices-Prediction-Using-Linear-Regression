# House-Prices-Prediction-Using-Linear-Regression

This project is mainly based on predicting house prices with a moderate data using the linear regression model on supervised learning for training machines. 

Taking a senerio, there are a lot of houses with different facilities like bathroom, bedroom, kitchens, garage ans os on. In order to predict the prices of those houses using a machine, you have to train the machine on a dataset containing those facilities then using it to make a prediction using a linear regression model. 

For example, you have a house at Los Angeles and another house at Saudi Arabia with a specific number of facilities, you will want to predict the price of the houses based on the number of the facilities, the location and how big the house is and instead of doing it manually, you do it with a machine and that's what this project is for.
